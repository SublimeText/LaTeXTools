LaTeXTools Plugin version 3.6.2 (2016-01-18) changelog:

Finally, a Script Builder! Control every step of the build process, or
integrate an existing build process into LaTeXTools.

New features:

- Script Builder (see above)
- Environment completion using latex-cwl (off by default)

Bugfixes:

- Ignore encoding errors when looking for tex root file
- Disable cwl-autocompletion if leading \ is escaped (R. Stein)
