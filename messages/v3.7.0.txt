LaTeXTools Plugin version 3.7.0 (2016-04-19) changelog:


New features:

- New Viewers:
  - Preview.app
  - Okular
- Improved bibiography parsing; should handle UTF-8 and most formatting
- Support for most citation commands and support for the various ways BibLaTeX has to add bibliographies
- Plugin system for handling viewers, builders, and bibliography file parsing
- Some internal improvements

Bugfixes:

- Internal bugfixes
