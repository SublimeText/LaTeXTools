LaTeXTools Plugin version 3.9.0 (2016-07-15) changelog:

Features:

- Jump to Anywhere
- New toggle quick panel
- Highlighting of errors, warnings and bad boxes
- Setting to control if PDF is opened on build
- Subfile support added for ref / cite completions
- \subfile{ now auto-completes file names
- Support for a coming revamp of the LaTeX syntax

Bugfixes:

- Traditional bibliography should support UTF-8 characters (#487)
- Traditional bibliography ignores spaces around the keyword (#366)
